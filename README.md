## Исключения

> Nonagod\Service\Exceptions

1. `ProgrammingException` - ошибки в логике (ошибки\неточности допущенные при написании кода)
2. `SystemException` - системные ошибки (ошибки конфигурации и т.п.)
3. `UserException` - ошибки пользователя (почти всегда должны быть доведены до пользователя)
